# Browserify experiment

## Versions I have checked

* Ubuntu
    * OS: Ubuntu 14.04 amd64
    * Node.js: 0.10.35
    * npm: 1.4.28
    * browserify: 8.0.1
* Arch
    * Node.js: 5.5.0
    * npm: 3.3.12
    * browserify: 13.0.0

## How to experiment

```sh
git clone git://github.com/kaosf/browserify-experiment.git
cd browserify-experiment
npm install -g browserify@13.0.0
cd a
./run.sh
#=> 6.00000000000001
```
